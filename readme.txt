== Installation ==

Requires Press Permit Core. Tested with WP 3.81 & Press Permit  2.1.44

1. Upload the press-permit-access-codes directory to the \`/wp-content/plugins/\` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the [pp-access-codes] shortcode to display the input field in the frontend.
